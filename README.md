## Objective
Use JavaScript **Fetch API** to get user data from a public API and display it on a webpage.

## API Used
`https://jsonplaceholder.typicode.com/users`

## Features
- Fetches user data using `fetch()` (async/await).
- Parses JSON and displays:
  - Name
  - Email
  - Full address (suite, street, city, zipcode)
- Shows loading, success, and error messages.
- Reload button to refetch data.
- Error handling using `try...catch`.

## How to Run
1. Open `index.html` in a browser (or use VS Code Live Server).
2. Ensure you have internet access.
3. Click **Reload Users** to fetch again.

## Concepts Practiced
- Fetch API
- Promises & async/await
- JSON parsing
- DOM manipulation
- Error handling in async code
